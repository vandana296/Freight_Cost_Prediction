# 🚚 Freight Cost Prediction System

An end-to-end Machine Learning project that predicts freight/shipping cost using real-world e-commerce data (Olist dataset).

---

## 📌 Project Overview

This project builds a complete ML pipeline to estimate freight cost based on product and logistics features such as weight, volume, delivery time, and distance.

---

## 🚀 Features

- 📊 Data preprocessing & feature engineering
- 🤖 Model training using XGBoost
- 📦 Prediction pipeline
- 🌐 FastAPI backend
- 🎨 Streamlit frontend
- 🐳 Docker support
- ☁️ Deployment ready

---

## 🧠 Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- FastAPI
- Streamlit
- Docker
